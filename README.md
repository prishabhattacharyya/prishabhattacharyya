## Hi There!👩🏽‍💻

I’m Prisha Bhattacharyya, a Master’s student in Natural Language Processing at UC Santa Cruz with a B.S. in Computer Science and a minor in Statistics. My work focuses on machine learning, large language models, and intelligent systems. I enjoy building impactful AI applications and continuously improving through hands-on projects.

## 📚 Featured Projects

<details>
<summary>🤖 AgentOS: Proactive Multi-Agent LLM System</summary>
<br>

- Built a proactive multi-agent system that predicts user intent from OS-level signals (Windows ETW)  
- Designed a Mixture-of-Experts routing layer achieving **87.1% routing accuracy**  
- Improved end-to-end task completion to **75% (vs. 52% baseline)**  
- Reduced time-to-assistance by **38%** via predictive context prefetching  
- Implemented shared episodic memory using ChromaDB for cross-agent context retrieval  

</details>

<details>
<summary>🔍 Machine-Generated Code Detection</summary>
<br>

- Fine-tuned CodeBERT on **500K+ multilingual code samples** for AI-generated vs human code detection  
- Achieved **99.3% accuracy / 0.993 macro-F1** with strong cross-domain generalization  
- Built scalable training and evaluation pipelines with GPU optimizations (fp16, gradient checkpointing)  


</details>

<details>
<summary>📈 StockPilot: RAG-Based Stock Recommender</summary>
<br>

- Built an LLM-driven recommendation system using LLaMA for personalized financial insights  
- Processed **7 years of SEC filings** into a structured, retrieval-augmented knowledge base  
- Outperformed S&P 500 benchmarks with **+63.8% (Value)** and **+70.8% (Income)** strategies  

</details>
<details>
<summary>🧠 Six Hats Conversational AI Chatbot</summary>
<br>

- Built a conversational AI chatbot using OpenAI GPT models to simulate six thinking personas for structured brainstorming  
- Improved idea diversity and decision-making clarity through multi-perspective agent interaction  
- Designed and implemented a multi-agent conversational pipeline with role-specific prompting and coordinated agent interaction  

</details>

<details>
<summary>🚀 NASA JPL: Satellite Cloud Removal with Deep Learning</summary>
<br>

- Reproduced and extended DSen2-CR (ResNet-based SAR-optical fusion) for cloud removal in Sentinel-2 imagery  
- Processed a **620GB dataset (157K+ image triplets)** using distributed training across 4 NVIDIA GPUs  
- Integrated Stable Diffusion inpainting to enhance reconstruction of cloud-obscured regions  

</details>
